Help Desk Dicas de um DEV

Olá! Seja bem vindo ao sistema Help Desk desenvolvido por mim no curso Desenvolvedor Full Stack.
Esse é um sistema para gerenciamento de chamados simulando um simples Help Desk desenvolvido em Angular 13 e Spring Boot. O projeto tem como objetivo principal abordar os conceitos de aplicações SPAs, Autenticação e Autorização com Tokens JWT e muito mais. O código fonte do projeto pode ser acessado clicando no link Github no menu lateral.

Fique a vontade para testar o sistema e deixar sua sugestão.
